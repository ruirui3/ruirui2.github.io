Export & Publish
Export your portfolio data as portfolio-data.json, then upload it to your GitHub repo root. The site auto-loads it on every fresh visit — no localStorage dependency.
🔄 How to update from another device
1. Export portfolio-data.json from this panel
2. Go to your GitHub repo → click portfolio-data.json
3. Click the ✏️ pencil icon → replace content → Commit changes
4. GitHub Actions will redeploy automatically (~1 min)
5. Any new visitor (or cleared cache) gets the latest data instantly
⚠️ localStorage vs JSON file
• localStorage = only on your current device/browser, lost on cache clear
• portfolio-data.json = lives on GitHub, loaded by every visitor, persists forever
• The site loads the JSON file first. If you've made admin changes locally, export and upload to make them permanent.
